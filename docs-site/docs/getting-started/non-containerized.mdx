Use the non-containerized path only when you need direct control over host Python, Perl, Java, MongoDB, and bioinformatics dependencies.

:::warning[Higher maintenance path]
This install mode is the most flexible, but it is also the easiest to break through dependency drift. Prefer Docker or Apptainer for first-time use when possible.
:::
