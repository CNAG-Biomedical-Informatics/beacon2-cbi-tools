**Beacon v2 CBI Tools** is an open-source toolkit for preparing Beacon v2 deployments with Beacon Friendly Format data.

:::note[Project name]
`CBI` refers to **CNAG Biomedical Informatics**. The project was formerly named **beacon2-ri-tools**.
:::

<div className="about-hero">
  <p>
    The project brings together Beacon v2 metadata validation, genomic data conversion,
    BFF JSON generation, MongoDB loading, and lightweight inspection utilities for
    research-oriented Beacon deployments.
  </p>
</div>

## Project Scope

<div className="about-grid">
  <article className="about-card">
    <span className="about-kicker">Core toolkit</span>
    <h3>Beacon data preparation</h3>
    <p>Validate metadata, convert VCF or SNP-array input into BFF genomic variation output, and load Beacon collections into MongoDB.</p>
  </article>
  <article className="about-card">
    <span className="about-kicker">Open source</span>
    <h3>Repository</h3>
    <p>The source code is available at <a href="https://github.com/CNAG-Biomedical-Informatics/beacon2-cbi-tools">CNAG-Biomedical-Informatics/beacon2-cbi-tools</a>.</p>
  </article>
  <article className="about-card">
    <span className="about-kicker">Documentation</span>
    <h3>Docusaurus docs</h3>
    <p>The documentation is maintained with <a href="https://docusaurus.io">Docusaurus</a> and organized around install paths, commands, workflows, outputs, and validation notes.</p>
  </article>
</div>
