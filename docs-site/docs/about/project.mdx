**Beacon v2 CBI Tools** is an open-source bioinformatics toolkit for preparing Beacon v2 data in Beacon Friendly Format. Its command-line interface is called **`bff-tools`**.

:::note[Project name]
**Beacon v2 CBI Tools** is the actively developed continuation of the original `beacon2-ri-tools` codebase, now developed at [CNAG Biomedical Informatics](https://www.cnag.eu) by its original developer.
:::

<div className="about-hero">
  <p>
    The project brings together Beacon v2 metadata validation, genomic data conversion,
    deterministic BFF JSON generation, and standalone inspection for research data.
  </p>
</div>
