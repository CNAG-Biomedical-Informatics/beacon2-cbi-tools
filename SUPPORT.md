# Support

Use the [documentation](https://cnag-biomedical-informatics.github.io/beacon2-cbi-tools/) for installation, annotation resources, data conversion, validation, and troubleshooting.

Open a [GitHub issue](https://github.com/CNAG-Biomedical-Informatics/beacon2-cbi-tools/issues) for reproducible defects, compatibility problems, or documentation gaps. Include the `bff-tools --version` output, execution environment, exact command, and sanitized diagnostic output. Questions about a downstream Beacon server should be directed to that implementation rather than this data-preparation toolkit.

Never post identifiable participant data, credentials, private URLs, or unrestricted genomic files in an issue.
