Help & Documentation

Welcome to the BFF Browser Help page. Here, you can find information on how to use the different features of the application.

Navigation

The navigation bar at the top of each page allows you to move between the Home page, Help page, and other sections. Use the menu to quickly access the Genomic Variations and Individuals sections, or return to Home.

Using the "View HTML by Path" Feature

The "View HTML by Path" option lets you input the path to an HTML file hosted on the server. Once submitted, the application will redirect you to that file, rendering its content with proper asset loading.

Individuals BFF Section

In the Individuals BFF section, you can:

  • See Individuals Example: View a sample JSON file displayed in a paginated table.
  • View Individuals JSON by Path: Provide the path to your own JSON file containing individuals' data. The application will read the file, process nested arrays into bullet lists, and display the data in a paginated table.

Data Display Mechanism

The application uses Python's pandas library to flatten JSON data and convert it into HTML tables. Nested arrays of objects within the JSON are rendered as bullet lists within table cells for clarity. DataTables is integrated to handle pagination, search, and sorting on the client side.

Security

For security reasons, file paths are validated to ensure safe and correct access. The application verifies that the provided path points to a valid file before processing it. While the system is designed to allow access from various directories, please ensure that the file paths you provide are accurate and that the files exist. This precaution helps protect the application from errors and unauthorized access.
