# Quick Start `bff-tools`

This guide provides a one-page cheat sheet for common commands:

```bash
# Display help for the tool
bin/bff-tools --help
```

```bash
# Display man for the tool
bin/bff-tools --man
```

```bash
# Convert VCF to BFF
bin/bff-tools vcf -i test/test_1000G.vcf.gz -p test/param.yaml
```
